##  Project Description

This project is an **AI Chatbot built using Natural Language Processing (NLP)** techniques in Python.
The chatbot is capable of understanding user queries and responding with relevant answers based on a predefined knowledge base (corpus).

It uses **NLTK (Natural Language Toolkit)** and **Machine Learning techniques (TF-IDF & Cosine Similarity)** to process and analyze text.

---

##  Objectives

* Build a simple AI chatbot using NLP
* Understand text preprocessing techniques
* Implement tokenization and lemmatization
* Generate responses based on user input

---

##  Technologies Used

* Python
* NLTK (Natural Language Processing)
* Scikit-learn (TF-IDF & Cosine Similarity)

---

##  Features

* Responds to basic user queries
* Handles greetings like *hi, hello, hey*
* Uses NLP techniques for understanding text
* Provides meaningful responses from dataset
* Simple and interactive command-line interface
